### 文獻名稱
Long-tail Internet photo reconstruction

- 論文來源URL: https://arxiv.org/abs/2604.22714

### 分類關鍵字
- 主題類別：Computer Vision
- 方法類別：Foundation Model, Dataset
- 任務類別：Reconstruction
- 難度：進階
- 關鍵字：3D reconstruction, Internet photos, long-tail scenes, MegaDepth-X, sparsity-aware sampling

### 一句話核心
這篇論文的核心發現是，只要先做出更乾淨的大型網路照片資料集，並用特別設計的稀疏取樣方式模擬真實世界那些照片很少又很雜亂的場景，就能明顯提升 3D 重建模型在困難網路場景中的表現。

### 為什麼要研究這個？（研究動機）
這篇論文在談的 3D 重建，意思是從多張 2D 照片推回場景的三維形狀與相機位置。論文特別關注的 long-tail（長尾）場景，是指網路上大多數不那麼有名的地標或建築，它們通常只有少量、角度分散、品質參差不齊的照片，不像熱門景點那樣有大量重疊又清楚的影像可用。

問題在於，傳統的 Structure-from-Motion, SfM（從多張照片估計相機位置與場景結構的方法）很依賴照片之間能找到穩定對應點，但長尾場景常常照片太少、視角差太大、甚至彼此幾乎沒有重疊，所以很容易失敗。即使是近年很強的 3D foundation model（大規模預訓練的 3D 基礎模型），也多半是在條件比較理想的資料上訓練，遇到這種真實世界又稀疏又混亂的網路照片時，還是容易重建出不一致的幾何結果。這件事很重要，因為現實世界真正大量存在的，其實正是這些「不完美資料很多、完整資料很少」的場景。

### 他們做了什麼？（研究方法）
研究者的想法很像是在訓練一個登山嚮導，不是只讓他走設備完善的觀光步道，而是故意把地圖切掉一大半、讓路線變得零碎，訓練他在資訊不足時也能找到方向。因為真正的長尾場景本來就很難拿到可靠的 3D 標註，所以他們反過來從那些本來就已經重建得很完整的大型網路地標資料中，人工抽出少量、稀疏、彼此弱連結的照片子集，拿來模擬長尾場景。

為了做到這件事，他們先建立一個新的資料集 MegaDepth-X（MD-X），從 MegaScenes 中挑出較可靠的場景，再用更嚴格的方法過濾掉人群、動態物體、錯誤配對與 doppelganger（看起來很像但其實不是同一個地方的混淆場景）問題，還加入更精細的深度圖修正。接著，他們提出 sparsity-aware sampling（稀疏感知取樣），就像不是隨便抽照片，而是故意挑那些分布比較開、連接比較弱、但局部還能勉強拼起來的視角，讓模型真的學會面對真實世界最難的情況。

### 驚人發現與具體數據（關鍵結果）
- 他們最後做出的 MD-X 資料集包含 1,865 個重建場景、總共約 44 萬張影像，規模明顯大於原本的 MegaDepth，代表這不是小修小補，而是一次大幅升級的訓練資源。
- 在困難的 hard 長尾測試場景上，π3 fine-tune 後的模型相較原始模型，AUC@5 從 36.93 提升到 47.93，代表相機姿態估計更準、更穩；VGGT 也從 29.10 提升到 41.49，進步非常明顯。
- 在同樣的 hard 場景中，π3 的平均點圖準確度誤差（Acc）從 0.101 降到 0.068，代表重建出來的 3D 幾何更接近真實場景；而且論文也展示它在真實長尾場景、甚至 COLMAP 幾乎完全失敗的情況下，仍能重建出較完整的結構。

### 這對我有什麼意義？（實際應用）
- 可以幫助用少量旅遊照片就重建古蹟、建築或景點的 3D 模型。
- 對數位典藏、文化資產保存很有價值，因為很多地方本來就沒有大量高品質影像。
- 能提升 AR、VR、地圖建模或視覺定位系統在真實世界稀疏資料下的穩定性。
- 也代表未來 3D AI 模型不只會在「資料很漂亮」的環境表現好，對一般人隨手拍的雜亂照片也可能更有用。

### 一句話核心
作者用更乾淨的網路照片資料集與稀疏感知取樣，讓 3D 模型更能應付真實世界少圖、亂圖、難配對的重建問題。

### 學習路徑
- 第一步：讀者應該先理解什麼背景？先理解 3D 重建、SfM 與多張照片之間的幾何關係。
- 第二步：接著理解哪個核心問題？理解為什麼網路上的長尾場景照片少、重疊低、雜訊高，會讓傳統方法與新模型都失準。

### 先備知識
- Structure-from-Motion（SfM）：從多張照片的對應點估計相機位置與場景三維結構的方法。
- 多視角幾何：研究不同相機視角之間如何對應到同一個真實世界結構。
- 深度圖：對影像中每個像素估計它離相機多遠的表示方式。
- Foundation Model：用大規模資料預訓練、可再微調到不同任務的基礎模型。
- 視圖圖（view graph）：把每張照片當成節點，照片之間能否穩定配對當成連線的圖結構。

### 重要名詞
1. **Long-tail scenes**：指網路上大量存在但照片很少、視角不均、品質不穩的場景。它重要的原因是這才是真實世界最常見的情況。它改善了過去研究過度偏向熱門、資料密集場景的盲點。
2. **MegaDepth-X (MD-X)**：作者建立的新資料集，來自較可靠的網路照片重建場景，並做了更嚴格的清理與深度修正。它重要的原因是模型需要可信的 3D supervision（監督訊號）才能學好。它改善了舊資料集規模不夠大或標註不夠乾淨的缺點。
3. **Sparsity-aware sampling**：一種專門模擬稀疏長尾場景的取樣方法。它重要的原因是模型不能只在密集、好配對的照片組合上訓練。它改善了隨機取樣或過度密集取樣無法反映真實困難場景的問題。
4. **Doppelganger problem**：不同地方卻長得很像，導致模型或重建系統把它們誤認成同一場景。它重要的原因是這會直接破壞相機姿態與幾何一致性。它改善了舊方法只靠局部特徵時容易誤配的缺點。

### 脈絡梳理
- 背景：這個領域原本在做的是從多張照片中重建 3D 場景，常用在地圖建模、文化保存、AR/VR 與視覺定位。
- 痛點：過去方法在熱門景點或高重疊影像上表現不錯，但真實網路照片常常又少又亂，甚至彼此幾乎看不到同一區域，導致重建失敗。這不是小問題，因為世界上大部分場景其實都屬於這種長尾分布。
- 作者的切入點：這篇論文不直接硬做難場景的標註，而是從原本可重建的大場景中，人工模擬出長尾條件來訓練模型，等於先在可控環境裡教模型面對真實困難。

### 方法流程
1. **挑出可靠母體場景**：研究者先從大型網路照片資料中挑選本來就有較穩定重建結果的場景，像是先找一批基礎資料比較完整的拼圖盒。
2. **清理錯誤與雜訊**：他們移除動態人群、錯誤重建與 doppelganger 混淆場景，就像先把拼圖盒裡混進來的別盒碎片挑掉。
3. **修正深度監督**：他們用多視角深度加上單眼深度先驗進一步過濾不合理像素，像是拿另一把尺重新確認哪些拼圖片真的對得上。
4. **模擬長尾照片分布**：他們不是隨便抽照片，而是故意挑選彼此距離較遠、關聯較弱、但局部還能連起來的視角，像是刻意用較少線索練習拼圖。
5. **微調 3D 模型**：接著用這些模擬出的困難訓練批次去微調 π3 與 VGGT，讓模型學會在資訊不足時仍維持幾何一致性。
6. **在真實困難場景驗證**：最後再用稀疏、混亂、容易失敗的真實網路照片測試，看模型是否真的比原始版本更穩。

### 文獻名稱
Structure-Guided Diffusion Model for EEG-Based Visual Cognition Reconstruction

### 文獻中文名稱
結構引導擴散模型：從腦波重建人類視覺認知影像

- 論文來源URL: https://arxiv.org/abs/2604.22649
- 抓取時間: 2026-04-27 20:03 (Asia/Taipei)

### 分類關鍵字
- 主題類別：Brain-Computer Interface / Computational Neuroscience
- 方法類別：Diffusion / CLIP / VAE / ControlNet
- 任務類別：Reconstruction / Generation / Prediction
- 難度：進階
- 關鍵字：EEG, Diffusion Model, Brain Decoding, ControlNet, CLIP, Visual Reconstruction

### 一句話核心
這篇論文的核心發現是，把「腦波中的結構線索」和「語意線索」一起拿來引導擴散模型，能比過去方法更穩定地重建出更像人腦真正理解內容的圖像。

### 為什麼要研究這個？（研究動機）
EEG（腦電圖）是用頭皮上的電極量測大腦活動的方法，優點是便宜、非侵入，而且時間解析度很高。Diffusion Model（擴散模型）則是一種從雜訊慢慢生成圖片的模型。這個領域想做的事很直接，就是從人的腦波反推出他看見了什麼，甚至理解了什麼。

但問題是，過去很多方法太依賴「這看起來像哪一類物體」這種粗略語意，卻沒真正抓住物體的結構，像是輪廓、部件關係、整體空間配置。對 EEG 來說，這尤其困難，因為腦波雜訊高、不同人的腦波差異也很大。結果就是，模型可能猜到大概是動物，卻畫不出像樣的形狀。這篇研究的重要性就在於，它想讓模型不只猜「是什麼」，還能更像人腦那樣理解「長什麼樣、怎麼組成」。 

### 他們做了什麼？（研究方法）
研究者的方法很像請一位畫家根據兩種提示作畫。第一種提示是「這張圖大概代表什麼意思」，第二種提示是「這個東西的結構草圖長怎樣」。他們先用 EEG 擷取腦波特徵，再把它對齊到 CLIP（把文字和圖片放進同一個語意空間的模型）的語意空間，讓模型知道受試者大概在想什麼。

接著，他們再做一件很關鍵的事，就像先畫草稿再上色一樣，先從 EEG 預測出低解析度的結構圖，再把這個結構圖送進 ControlNet 去約束擴散生成。同時，語意訊息則透過 IP-Adapter 注入生成過程。最後的圖像不是只靠模糊語意亂猜，而是在「語意」和「結構」雙重限制下生成，因此更容易畫出既像、又有條理的結果。

### 驚人發現與具體數據（關鍵結果）
- 在 Tangram（Kilogram）資料集上，SGDM 的 CLIP Score 達到 0.396，明顯高於 Base 的 0.222，代表生成圖和目標圖在高階語意上更接近。
- 在 THINGS 資料集上，SGDM 的 SwAV FID 為 23.979，優於 Base 的 24.408，也優於 BrainVis 和 DreamDiffusion，表示整體表徵品質更好。
- 在結構重建上，SGDM 的 IoU 顯著高於比較方法，Kilogram 的提升達統計顯著（p=0.013），而最終圖像相似度在 Kilogram 與 THINGS 也都顯著更好（分別為 p=0.021 與 p=0.037）。

### 這對我有什麼意義？（實際應用）
- 未來可用在更自然的腦機介面，幫助系統更準確理解人腦中的視覺想法。
- 對失語或無法表達的人，可能成為輔助溝通的新方向。
- 可用來研究人類如何從局部形狀一路形成完整語意，幫助神經科學理解視覺認知。
- 在神經復健、意識評估與個人化認知分析上，也有潛在應用價值。

### 學習路徑

### 先備知識
- EEG：用頭皮電極記錄大腦電活動，時間解析度高，但訊號雜訊也高。
- CLIP：把文字和圖片映射到同一個語意空間，方便比較兩者是否表達相近意思。
- Diffusion Model：先從雜訊開始，再一步步去噪生成圖片的模型。
- ControlNet：在擴散模型中加入額外結構條件，幫助生成結果遵守空間布局。
- VAE：把資料壓縮成潛在表示再重建，常用來萃取影像的結構特徵。

### 重要名詞
1. **SGDM**：這是作者提出的整體框架，目標是從 EEG 重建更符合人類認知的影像。它重要的原因是把結構與語意一起納入生成流程。它改善了舊方法只會抓大概語意、卻畫不準形狀的問題。
2. **Structure Prediction Module**：這個模組會先從 EEG 預測出粗略的視覺結構圖。它重要的原因是等於先幫生成模型打草稿。它改善了舊方法缺少明確空間約束、容易生成模糊輪廓的缺點。
3. **CLIP-based EEG Semantic Alignment**：這是把 EEG 特徵對齊到 CLIP 語意空間的做法。它重要的原因是讓模型知道腦波大致對應到什麼視覺概念。它改善了 EEG 和影像語意空間落差太大、訓練不穩的問題。
4. **ControlNet + IP-Adapter 雙重條件生成**：ControlNet 管結構，IP-Adapter 管語意，兩者一起控制擴散生成。它重要的原因是把「像什麼」與「長怎樣」分開但協同處理。它改善了過去只能單一路徑控制生成、平衡不好的一大缺點。

### 脈絡梳理
- 背景：這個領域想把腦訊號轉成圖像，幫助理解人類視覺認知，也可能應用在腦機介面與輔助溝通。
- 痛點：過去 EEG 解碼方法常只能抓到類別語意，難以保留局部結構、整體布局與個體差異，這會讓生成結果看起來像是「猜中類別但畫錯內容」。
- 作者的切入點：這篇論文最大的差別，是明確把「結構資訊」獨立出來建模，先從腦波預測結構，再把結構和語意一起餵給擴散模型。

### 方法流程
1. **建立共同語言**：研究者先用 CLIP 把文字描述和圖片放進同一個語意空間，就像先定好一套大家都懂的翻譯字典。
2. **讀懂腦波意思**：再用 EEG 編碼器把腦波特徵轉成這套語意字典中的表示，等於把大腦訊號翻成模型能懂的意思。
3. **先畫結構草圖**：接著用結構預測模組，從 EEG 生成一張粗略的結構圖，像畫畫前先打底稿。
4. **雙條件引導生成**：生成圖片時，語意表示透過 IP-Adapter 告訴模型「內容像什麼」，結構圖透過 ControlNet 告訴模型「空間怎麼擺」。
5. **比較生成品質**：最後用 CLIP Score、SSIM、SwAV FID、IoU 等指標，檢查生成結果在語意、結構與整體品質上是否優於基線方法。

Are Natural-Domain Foundation Models Effective for Accelerated Cardiac MRI Reconstruction ?

自然影像領域的基礎模型，真的適合用來做加速心臟 MRI 重建嗎？

### 分類關鍵字
- 主題類別：Medical Imaging
- 方法類別：Foundation Model / Transformer
- 任務類別：Reconstruction

### 一句話核心
這篇論文發現，雖然專門為 MRI 訓練的模型在一般情況下還是最強，但在跨資料域或重建更困難的情況下，來自自然影像的 Foundation Model 也能提供很有競爭力的結構先驗，甚至有時會更好。

### 為什麼要研究這個？（研究動機）
先講幾個核心名詞。MRI 是磁振造影，可以不靠輻射看到身體內部結構；重建是指把儀器量到的原始訊號還原成影像；加速 MRI 則是故意少量測一些資料，縮短掃描時間，但代價是影像更難還原。Foundation Model（基礎模型）則是先在超大量資料上預訓練，學到可轉移能力的大模型。

這篇研究想解決的痛點很直接，心臟 MRI 很有用，但掃描慢，病人容易不舒服，也更容易因為移動而讓影像變糊。所以大家會想少量測一些資料來加快掃描，但這會讓重建變成很難的「拼圖問題」。過去很多深度學習方法雖然很強，但往往只能在自己熟悉的資料條件下表現好，一換掃描條件、機器、器官或資料分布，效果就可能掉很多。

研究者因此想問一個很有意思的問題，如果我們不用完全為 MRI 量身打造的新模型，而是借用像 CLIP、DINOv2、BiomedCLIP 這種已經在大規模資料上學過很多結構知識的模型，能不能把它們當成一種「有見過很多世界的顧問」，幫忙把缺資料的 MRI 影像補得更合理？

### 他們做了什麼？（研究方法）
你可以把整個 MRI 重建過程想像成在修一張缺角很多的拼圖。一般方法會找一個很熟這類拼圖的專家，反覆修補，直到拼圖看起來最像真的。這篇論文的做法則像是在每一輪修補時，都請一位見多識廣的圖像老師來看一下，告訴系統「這裡的結構大概應該長什麼樣」。

具體來說，他們把 CLIP、DINOv2、BiomedCLIP 這些已經預訓練好的視覺 Transformer 編碼器固定住，不重新訓練它們，只把它們當成特徵提取器，嵌進一個反覆迭代的 MRI 重建架構裡。每一輪先做資料一致性檢查，確保結果沒有偏離真實量測，再利用這些 Foundation Model 提供的結構特徵來微調影像，就像一邊核對原始證據，一邊請高手幫你補全細節。

### 驚人發現與具體數據（關鍵結果）
- 在同資料域的心臟 MRI 測試中，專門為 MRI 重建訓練的 E2E-VarNet 仍然最好。例如在 ×4 加速下，SSIM（結構相似度，越高越好）達到 0.9676，高於 CMR-CLIP 的 0.9585，代表專用模型在標準條件下還是更穩。
- 但當加速更高、問題更難時，Foundation Model 方法仍然很有競爭力。比如在 ×10 加速下，CMR-BiomedCLIP 的 SSIM 為 0.9240，和 E2E-VarNet 的 0.9417 差距已經縮小，表示在資料更少、重建更難時，這些預訓練表示有實力撐住表現。
- 論文最重要的觀察是，到了跨資料域測試和更強烈欠採樣的情境，Foundation Model 與專用模型的差距明顯縮小，某些情況下甚至反超。也就是說，這些從自然影像學來的結構知識，真的能轉移到醫學影像重建，而且在分布改變時更有韌性。

### 這對我有什麼意義？（實際應用）
- 未來有機會讓心臟 MRI 掃描更快，減少病人等待時間與不適感。
- 對醫院來說，若模型更能適應不同設備或資料來源，就比較不需要每換一種場景就重訓一次。
- 這代表大型通用視覺模型不只會看自然照片，也可能成為醫學影像重建的重要基礎工具。
- 在資料少、跨醫院、跨器官或跨掃描條件的情境下，這類方法可能比傳統專用模型更實用。

### 一句話核心
這篇論文證明，凍結的視覺 Foundation Model 能成為加速 MRI 重建的可轉移先驗，特別在跨資料域與高加速情境下更有價值。

### 學習路徑
- 第一步：先理解 MRI 為什麼掃描慢，以及為什麼少量測資料後重建會變難。
- 第二步：接著理解深度學習重建模型為什麼常在跨設備、跨器官或跨資料域時失效。
- 第三步：最後用「通用視覺知識能不能當作影像結構先驗」的角度看這篇方法與結果。

### 先備知識
- [k-space]：MRI 不是直接拍出圖片，而是先在頻率域收集訊號，再轉回影像。
- [Undersampling]：為了加快掃描，只量測部分資料，但這會讓重建資訊不足。
- [Inverse Problem]：從不完整或有雜訊的觀測資料反推出真實影像，是一類本質上不容易的問題。
- [Unrolled Network]：把傳統反覆優化流程展開成多步神經網路，讓每一步都學會更好的修正方式。
- [Foundation Model]：先在大規模資料上預訓練，再把學到的表示能力轉用到其他任務的大模型。

### 重要名詞
1. **Accelerated MRI Reconstruction**：這是從欠採樣的 MRI 原始訊號中重建出可用影像的任務。它重要的原因是 MRI 掃描很慢，縮短掃描時間能減少病人不適與運動偽影。它改善了傳統完整掃描太耗時的缺點。
2. **E2E-VarNet**：這是 MRI 重建領域很強的專用型 unrolled 網路基線。它重要的原因是能把物理模型與學習到的影像先驗結合，常在標準測試中表現最好。它改善了較舊方法只靠手工先驗、重建品質有限的缺點。
3. **CLIP / DINOv2 / BiomedCLIP**：這些是不同來源預訓練的視覺 Foundation Model，分別代表自然影像語意對齊、自監督視覺表示與生醫圖文表示。它們重要的原因是能提供可轉移的結構特徵。它們改善了只靠小型、特定資料集訓練時泛化不足的缺點。
4. **Data Consistency**：這是在每次重建更新時，把結果拉回真實量測資料約束的一步。它重要的原因是可以避免模型只靠想像補圖，卻偏離實際量測。它改善了純影像生成式方法可能不符合物理量測的缺點。

### 脈絡梳理
- 背景：這個領域原本就在研究如何從較少的 MRI 量測資料中重建清楚影像，因為臨床上大家都想讓掃描更快、更穩，也更能適應不同病人與儀器。
- 痛點：過去方法雖然在熟悉資料集上很強，但常依賴特定 MRI 資料分布，一遇到不同器官、不同掃描條件或更高加速比，表現就可能明顯下降。這不是小問題，因為臨床環境本來就充滿跨設備、跨院所與跨任務差異。
- 作者的切入點：這篇論文最大的不同，是不把 Foundation Model 拿來直接做高階分類任務，而是把凍結的視覺編碼器當成重建過程中的「結構顧問」，測試它們能不能提供更有泛化性的先驗。

### 方法流程
1. [先拿到不完整的 MRI 訊號]：研究者先從多線圈心臟 MRI 的欠採樣 k-space 開始，這就像手上只有拼圖的一部分，必須想辦法補回整張圖。
2. [估計量測規則]：系統先估計 coil sensitivity map，等於先弄懂每個感測器是從什麼角度在看影像，避免後面補圖時失去物理依據。
3. [做一輪資料一致性修正]：每個 cascade 都先檢查目前重建結果有沒有偏離真實量測，像是每修一次拼圖都先對照手上確定存在的碎片。
4. [請 Foundation Model 提供結構提示]：把中間重建結果轉成符合預訓練模型輸入的格式，送進 CLIP、DINOv2 或 BiomedCLIP，提取多層特徵，就像請見過大量圖像的老師指出哪裡的結構比較合理。
5. [用輕量解碼器把提示變回重建修正]：系統把多層特徵融合後，經過階層式解碼器輸出新的影像修正，再回到下一輪迭代。
6. [反覆多輪直到輸出最終影像]：整個流程重複多個 cascades，讓結果同時符合量測資料，也吸收預訓練表示中的結構知識。

# Multivariate Gaussian NeRF for Wide Field-of-View Ultrasound Reconstruction

Multivariate Gaussian NeRF for Wide Field-of-View Ultrasound Reconstruction

多變量高斯 NeRF 用於廣域超音波重建

### 一句話核心
用**多變量高斯分佈**模擬超音波探頭的**發散幾何**，當個天然的反鋸齒濾波器，消除拼接縫並產生全新視角。

### 秒懂表格
| 問題 | 秒懂回答 |
|---|---|
| 這篇在解決什麼？ | 現有超音波拼接方法會有**拼接縫**和**失真**，因為忽略了超音波探頭發散的物理特性。 |
| 作者怎麼做？ | 用**多變量 3D 高斯**替換傳統「單薄射線」，並根據**深度**調整採樣範圍，模擬真實發散光束。 |
| 結果有多好？ | **消除拼接偽影**，結構邊緣更銳利，且訓練速度**快 10-20 倍**。 |
| 為什麼重要？ | 能讓超音波影像從**單點掃描**升級為**互動式全景診斷工具**，大幅擴增臨床視覺範圍。 |

### 為什麼要研究這個？（研究動機）
- **臨床診斷需要全景視野**：單個超音波掃描視野有限，需要拼接多組才能看到完整器官結構。
- **傳統方法有根本缺陷**：現有方法假設超音波是平行射線，但實際探頭（特別是凸型探頭）的光束會**隨深度發散**。
- **忽略發散會造成嚴重失真**：越遠處解析度越低、出現偽影、各方向解析度不一致。
- **NeRF 技術可以解決**：神經渲染技術能建模物理傳播路徑，本論文想把它應用到超音波領域。

### 他們做了什麼？（研究方法）
- **重新定義凸型探頭幾何**：不用平行射線，改用**極坐標 + 錐形截斷**來模擬發散光束。
- **從點到多變量高斯**：不再把採樣區當成一個點，而是用**3D 多變量高斯分佈**來描述它的體積範圍。
- **各向異性變異數設計**：針對超音波的三個方向（射線方向、側向、深度方向）分別設計變異數。

### 30 秒看懂這篇論文
- **解決什麼問題：** 從稀疏、雜訊且低解析度的觀測數據中，還原符合物理定律的高解析度完整場。

### 先備知識
- **PDE（偏微分方程）：** 描述物理系統演化規律的數學方程式，已知參數與邊界條件後可用正向求解器推演系統狀態。
- **擴散模型：** 一種生成式模型，透過逐步加雜訊再學習去雜訊的過程，能從雜訊中生成符合分佈的樣本。
- **潛在變量（latent variable）：** 低維度的隱藏表示，能解碼回高維度的觀測數據，這裡用來嵌入物理參數。
- **頻譜方法：** 將解表示为全球基函式的展開，透過傅立葉係數轉換可高效地處理微分操作。
- **逆問題：** 從觀測數據反推系統參數或初始條件的問題，通常病態且需要正則化。

### 重要名詞
1. **[LatentPDE]**
   - 白話描述：將 PDE 係數作為潛在變量表示的擴散模型框架。
   - 意義：解決了傳統方法無法同時保證物理一致與不確定性量化的問題。
2. **[可微分光譜求解器]**
   - 白話描述：一個可以對其輸入進行微分計算的光譜求解器，用於將潛在變量解碼為物理場。
   - 意義：讓物理定律成為解碼過程的結構約束，而非僅是軟規則化。
3. **[條件化表示]**
   - 白話描述：四通道空間表示，包含遮蔽場、二值遮蔽、距離觀測地圖與觀測密度地圖。
   - 意義：讓網絡分辨未觀測的零值與真實測量，並暴露感測器模式的空間結構。
4. **[振幅後驗（MAP）估計]**
   - 白話描述：透過最小化正則化損失函數，直接求得最可能的潛在變量。
   - 意義：不需訓練數據就能配合觀測，但每樣本優化較慢。
5. **[推論編碼器（amortized encoder）]**
   - 白話描述：訓練好的神經網絡，一次前向傳播就能預測潛在變量。
   - 意義：推論極快，但可能對訓練時 unseen 的遮蔽模式泛化不佳。
6. **[擴散去雜訊模型]**
   - 白話描述：在潛在空間進行去雜訊的擴散過程，學習符合物理定律的潛在先驗分佈。
   - 意義：提供 calibrated 的不確定性量化，捕捉後驗分佈。
7. **[物理引導（physics guidance）]**
   - 白話描述：在擴散後向步驟中，使用遮蔽殘差進行梯度下降來修正潛在預測。
   - 意義：確保最終重建結果符合物理定律，而非僅是統計平均。

### 研究動機
- **背景：** 物理系統逆向模擬需要從觀測數據推演完整場，應用於天氣預報、醫學成像等高風險場景。
- **痛點：** 現有方法（如 EnKF、3D-Var、PINNs）要麼無法量化不確定性，要麼過度平滑遺失重要動態。
- **目標：** 建立一個能同時保證物理一致性、不確定性量化，且能從稀疏觀測中還原高解析度場的框架。

### 方法流程
1. **[條件化表示生成]**：將稀疏觀測與初始條件轉換為四通道空間表示。
2. **[潛在變量初始估計]**：使用 MAP 優化或編碼器推論獲得初始潛在變量。
3. **[擴散去雜訊過程]**：透過去雜訊擴散模型在潛在空間進行 refine。
4. **[物理引導修正]**：在後向步驟中使用物理約束修正潛在預測。
5. **[可微分光譜解碼]**：將最終潛在變量解碼為高解析度、物理一致的場。
